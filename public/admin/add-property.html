Novo imóvel

Cadastro operacional completo para alimentar o site e a rotina da equipe.

Como cadastrar Voltar para imóveis
Fluxo recomendado: comece por dados básicos e endereço, depois valores e características. Finalize com fotos, detalhes e descrição para garantir um card completo no site.

Dados Básicos

Base do cadastro, organização comercial e visibilidade do imóvel no portal.

Endereço

Campos estruturados para facilitar localização, busca futura e integração visual com o portal.

Valores

Cadastro financeiro comercial, com opções adicionais para financiamento, FGTS e ágio.

Características

Estrutura física do imóvel com os mesmos detalhes que o time consulta no atendimento.

Detalhes

Checklist mais rico para não depender de observações soltas no time comercial.

Descrição

Texto comercial com apoio de preenchimento automático para agilizar o time.

Geração automática de descrição

Use o botão para criar uma base comercial com as informações que já foram preenchidas. Depois refine o texto antes de salvar.

Fotos

Escolha a capa do card e mantenha a galeria em uma ordem lógica para o cliente.

Sugestão de ordem: fachada ou área comum, sala, cozinha, quartos, banheiros e lazer. Use “Definir capa” para escolher a imagem principal do card.
