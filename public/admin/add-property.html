Como cadastrar Voltar para imóveis
Fluxo recomendado: preencha primeiro os dados principais do imóvel, depois valores e por último as fotos. Para locação, informe aluguel bruto e desconto de pontualidade que o painel calcula o valor líquido automaticamente.
Identificação do imóvel
Esses dados aparecem no card, nos detalhes e ajudam a equipe a localizar o imóvel depois.
Valores comerciais
Para aluguel, informe bruto e desconto. O campo preço vira o valor líquido. Para venda, preencha diretamente o preço final.
Características
Preencha a estrutura física do imóvel. Se algum dado não existir, pode deixar zero.

Somente imóveis com status “Ativo” aparecem no site público.

Localização e texto comercial
Use a descrição para colocar diferenciais, lazer, regras da locação e condições especiais.

Ordem sugerida: fachada ou área comum primeiro, depois sala, cozinha, quartos, banheiro e lazer. Use o botão “Definir capa” para escolher a imagem principal do card.

Detalhes para a equipe
Esses campos ajudam o time a organizar melhor cada imóvel sem depender de observações soltas.
